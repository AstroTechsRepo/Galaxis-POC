# Galaxis — Présentation soutenance 2025/2026

## Démarrage

Ouvrez `index.html` dans Chrome ou Firefox.
Cliquez sur **Commencer ▶** pour démarrer la présentation.

## Contrôles clavier

| Touche | Action |
|---|---|
| `→` / `Page Down` / `Espace` | Slide suivante |
| `←` / `Page Up` | Slide précédente |
| `Home` | Première slide |
| `End` | Dernière slide (15) |
| `O` ou `Esc` | Ouvrir / fermer la vue d'ensemble |
| `F` | Basculer plein écran |
| `B` ou `.` | Écran noir (pause) |

## Compatibilité télécommande

La présentation est compatible avec les télécommandes Logitech, Kensington et autres qui envoient les codes standards `PageUp` / `PageDown` ou `←` / `→`.

## Structure

- `index.html` : orchestrateur de présentation
- `slides/01.html` à `slides/15.html` : les 15 slides du parcours principal
- `slides/16.html` : annexe (accessible uniquement via overview)

## Vue d'ensemble (overview)

Appuyez sur `O` pour afficher toutes les slides en grille. La slide annexe 16 est dans une section séparée "Annexes — pour les questions du jury".

Cliquez sur une miniature pour y aller directement.
