# Documentation PROJET — Galaxis POC

> **Audience** : jury ESGI, chef de projet, sponsor · **Version** : v1.0-soutenance · **Édition** : mai 2026

Cette documentation raconte **pourquoi** et **comment** le projet Galaxis a été mené : le contexte marché, les personas, les choix de cadrage, le déroulé, les difficultés, la roadmap.

Elle est volontairement narrative — pas un dump de specs techniques — pour permettre à un lecteur non développeur de comprendre la genèse et la trajectoire du produit.

## Pour quoi cette doc ?

- **Le jury** : comprendre le projet en 30 minutes sans lire le code
- **Un investisseur ou un partenaire** : évaluer la maturité de cadrage et la trajectoire
- **Un futur chef de projet** : reprendre le pilotage sans perdre le contexte
- **L'auteur lui-même dans 6 mois** : se rappeler **pourquoi** telle décision a été prise
