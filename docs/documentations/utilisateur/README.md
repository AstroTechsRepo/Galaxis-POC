# 📕 Guide UTILISATEUR — Galaxis

> Bienvenue sur Galaxis ! Ce guide est fait **pour vous** : le ou la dirigeant·e de TPE qui veut reprendre la main sur ses outils, sans appeler son neveu informaticien.

---

## 🎯 À qui s'adresse ce guide ?

- Vous dirigez ou co-dirigez une **petite équipe** (1 à 20 personnes)
- Vous gérez vous-même les comptes, les accès, les mots de passe
- Vous n'êtes **pas** un·e expert·e technique — et ce n'est pas grave
- Vous voulez juste que **ça marche**, sans surprise

Si c'est vous, vous êtes au bon endroit.

## 💡 Comment ce guide est conçu

- **Pas de jargon technique** — quand un mot compliqué apparaît, il est expliqué tout de suite et noté dans le [glossaire](./08-glossaire.md)
- **Phrases courtes** — vous lirez plus vite, vous comprendrez plus vite
- **Étapes numérotées** — vous pouvez suivre le doigt sur l'écran
- **Encarts colorés** :
  - 💡 **Astuce** : un raccourci qui fait gagner du temps
  - ⚠️ **Attention** : un piège fréquent à éviter
  - ✅ **C'est bon** : confirmation qu'une étape est réussie
